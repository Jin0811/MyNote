# 笔记整理


[CustomDirectives]：Vue自定义全局指令、按钮权限控制

[ElementMessage]：ElementUI的消息组件二次封装

[VuePreviewPDF]：Vue 项目当中使用 PDF.js 来预览 PDF 文件
